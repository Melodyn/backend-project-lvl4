## Установка и запуск

Setup:
```sh
make setup # при первой установке
```

После установки зависимостей и прав на исполняемый файл, будет создан `development.env`, где можно указать собственный `ROLLBAR_PSI_TOKEN`.

Run:
```sh
make # или `make start`
```
